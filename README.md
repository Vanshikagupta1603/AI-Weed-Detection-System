# AI-Weed-Detection-System

AI-powered Precision Agriculture System using YOLOv8.

## Features

- Crop Detection
- Weed Detection
- Crop Count
- Weed Count
- Weed Density Analysis
- CSV Report Generation

## Tech Stack

- Python
- YOLOv8
- OpenCV
- Streamlit
- Pandas

## Dataset

6552 annotated agricultural images

## Performance

- mAP50 > 80%
- Precision: 78.6%
- Recall: 78.6%
